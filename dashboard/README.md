# Time dashboard

The Tauri 2 + React dashboard half of Time. See the [root README](../README.md)
for what Time is, how the two halves fit together, and how to run everything.

Quick reference: `npm install` then `npm run tauri dev` (uses the live DB;
point at a demo DB with debug-only `TIME_DB_PATH` — see `scripts/make_demo_db.py`).
Tests: `npx vitest run` · typecheck: `npx tsc --noEmit`.
